前苏格拉底哲学家

泰勒斯

  1. 智慧广博，卖榨油机的故事
  2. 西方哲学的开端，万物皆生于水
  3. 对世界本质“一”的思考

阿那克西曼德

  1. 世界来源于不确定的无限制的实在
  2. 万物是在永恒的运动中产生的
  3. 人类是由其他生物进化而来的
  4. 同时并存着多个世界和宇宙，而且终将毁灭

阿那克西米尼

  1. 气是万物的本源
  2. 事物的不同在于气的凝聚程度的不同，质的差异来源于量的差异，事物状态的变化在于气的收缩或凝聚
  3. 米利都学派第一次提出了关于事物最终本性的思考

毕达哥拉斯

  1. 事物是由数构成的，数的原则就是所有事物的原则
  2. 灵魂的三重区分
  3. 音乐与数的关系，音乐与人生活的和谐
  4. “形式”的概念，对柏拉图“理念”的影响

赫拉克利特

  1. 一切都处于流变之中，世界的多样性统一于火的变化，事物只是火的不同形态
  2. 变化过程是神的普遍理性(logos)的产物，万事万物都分享了神的普遍理性或普遍规律
  3. 斗争正是变化的本质，多在一中统一，一在多中体现

巴门尼德

  1. 存在是单一的、不变的、永恒的东西
  2. 世界是不可分的
  3. 真理之路和意见之路

芝诺

  1. 我们的感官没有为我们提供可靠的知识
  2. 四个悖论（运动场、追龟、飞矢不动、运动的相对性）
  3. 重申了巴门尼德“变化和运动乃是幻象”的观点

恩培多克勒

  1. 存在是永生的、不可毁灭的
  2. 存在不是一而是多
  3. 物质由土、气、火、水四种元素构成，四种元素在“爱”和“恨”两种力的循环作用下带来事物形态的变化

阿那克萨戈拉

  1. 提出了和物质相区分的“心灵”(nous)概念，心灵无处不在
  2. 实在的每个部分都和实在的全体有着同样的成分
  3. 理性使得这个世界上的事物井然有序并具有精妙复杂的结构
  4. 原始物质形成事物是通过在心灵力量作用下的分离，这种分离永不间断地进行下去

留基波和德谟克利特

  1. 提出“原子论”，即所有事物都是由运动在虚空中的原子构成的
  2. 自然只包含了真空的空间和无限数量的绝对致密的原子，原子不断运动形成土、气、火、水等元素和我们看到的物体
  3. 思想也是原子的运动，区分了感性知觉和理性知觉
  4. 区分了真实的知识和暧昧的知识，我们通过感官不能知道任何确切的真理
  5. 提出一套伦理学的规则，认为生活最令人向往的目标是快乐

智者和苏格拉底

普罗泰戈拉

  1. 最有影响力的智者，迫使雅典人对自己的行为方式进行反思，提出人为(nomos)和自然(physis)的区别，坚持怀疑主义的观点
  2. 人是万物的尺度，我们没有办法区分一个事物的现象和它的实在，也不可能获得绝对的科学知识
  3. 道德判断也是相对的，法律和道德是基于习俗，没有方法能够断定它们的对错
  4. 为了社会和平有序，每个人都应该遵守城邦的法律，年轻人应该接受和教育社会的传统
  5. 我们不能确定诸神的存在和本质，不过并不妨碍我们对于神的崇拜

高尔吉亚

  1. 拒绝承认任何真理的存在
  2. 无物存在，存在也不能被认识，认识也不能被传达

塞拉西马柯

  1. 不正义比正义的生活更可取，只有傻子才追求正义，正义只会导致软弱
  2. 有力即有理，法律是由统治者为了自己的利益制定的，“正当”仅仅是更强大集团的利益
